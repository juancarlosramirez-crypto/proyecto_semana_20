# Sistema de Gestión de Alumnos - Frontend

Este es el frontend de la aplicación desarrollado con **React** y **Vite** para el proyecto de "Desarrollo de interfaces web modernas con React". La aplicación permite listar, ver detalles y gestionar alumnos consumiendo una API REST en producción.

## 🚀 Características
- Componentes funcionales modernos.
- Uso de React Hooks para el manejo del estado y ciclo de vida (`useState`, `useEffect`).
- Renderizado dinámico de datos mediante `props`.
- Consumo de servicios API asíncronos.

## 🛠️ Requisitos Previos
Antes de iniciar, asegúrate de tener instalado:
- [Node.js](https://nodejs.org/) (Versión 18 o superior recomendada)
- [npm](https://www.npmjs.com/)
